With this easy-to-use tool, you can convert any date from the Gregorian calendar to its equivalent date in the Bikram Sambat calendar and vice versa. The plugin provides a seamless way to work with both calendars, making it ideal for individuals and developers working with Nepali dates.

Contributions from wprefers ensure that this plugin stays up-to-date and reliable.

Enjoy the convenience and accuracy of the Nepali Date Wizard for all your date conversion needs.

== Description ==
The plugin allows you to effortlessly convert dates between AD (Gregorian) and BS (Bikram Sambat) calendars.

Features :

1. Date Conversion: Effortlessly convert dates between AD (Gregorian) and BS (Bikram Sambat) calendars.
2. Two-Way Conversion: Convert any date from the Gregorian calendar to its equivalent date in the Bikram Sambat calendar and vice versa.
3. Ideal for Nepali Dates: Perfect for individuals and developers working with Nepali dates, making it convenient to handle date conversions.

== Installation ==
#### From within Wordpress
1. Visit 'Plugins  > Add New'
2. Search 'Nepali Date Wizard'
3. Install and Activate the plugin.
4. Create post/page and add the shortcode `[nepali-date-wizard]`

#### Manually
1. Download and unzip the file.
2. Upload the folder Nepali Date Wizard plugin into 'wp-content/plugins/' folder.
3. Activate from the plugin page.
4. Create post/page and add the shortcode `[nepali-date-wizard]`

== Frequently Asked Questions ==
= How does the plugin work? =
The plugin simplifies switching between the Gregorian (AD) and Bikram Sambat (BS) calendars. It has two methods, convertAdToBs and convertBsToAd, for date conversions.
The plugin ensures accurate results by validating input dates and using predefined configurations.
With this tool, users can easily convert any AD date to its BS equivalent and vice versa, making it perfect for working with Nepali dates.
